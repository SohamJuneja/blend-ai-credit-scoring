### üöÄ Getting Started

1. **Clone the Repository**

   ```bash
   git clone <repository-url>
   cd blend-ui-1
   ```

2. **Install Dependencies**

   ```bash
   npm install
   ```

3. **Run Development Server**

   ```bash
   npm run dev
   ```

4. **Open in Browser**
   Navigate to `http://localhost:3000`
